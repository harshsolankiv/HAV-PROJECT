Easy Trade


In this project we have developed website based on Django framework using which user can login register themself. User can read the live prices and fundamentals of particular stock. Also they can see the graphical form of data in candlestick. User can see the indices of NSE and BSE for trading. Top gainer and loser are also updated as per market time. User can make watchlist of the stock in which they want to invest and can note there daily performance. User can see the monthly analysis of the stock of their historical data and see the probable future value of it. New user to trading can learn trading and investing in stock. User can also see the predicated prices of stock which is using LSTM model. Also the heatmap of Nifty50 from 2005 to 2021 can be analyzed. User will get time to time hot news about stock market on news section.


Technology used:
  • Python
  • Django
  • HTML
  • CSS
  • JavaScript
  • SQL lite
  • LSTM model
  • Streamlit
